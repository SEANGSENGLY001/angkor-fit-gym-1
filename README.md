# AngkorFit - Train Like a Pro

AngkorFit is a premium fitness center website template designed for a modern gym in Phnom Penh, Cambodia. It features a responsive, high-performance landing page that highlights classes, trainers, membership plans, and schedules, inspired by Khmer strength and resilience.

## Features

- **Modern & Responsive Design**: Fully responsive layout that looks great on mobile, tablet, and desktop.
- **Dynamic Classes & Schedule**: High-quality class showcases (Temple Yoga, Khmer HIIT, Naga Spin, Stone Strength) and a detailed weekly timetable.
- **Membership & Pricing**: Clear, tier-based membership plans (Bronze, Gold, Temple) without lock-in contracts.
- **Trainer Profiles**: Showcasing elite coaches to build trust and credibility.
- **Contact & Location**: Integrated contact form and Google Maps embed for easy navigation.
- **SEO & Accessibility Optimized**: Built with semantic HTML, ARIA labels, and meta tags for optimal search engine performance and accessibility.

## Tech Stack

- **HTML5**: Semantic and accessible markup.
- **CSS3**: Custom styling, responsive Grid/Flexbox layouts, and smooth animations.
- **Vanilla JavaScript**: Interactive UI elements including scroll spy, mobile navigation, sticky headers, and form validation.

## Getting Started

1. Clone the repository.
2. Open `index.html` in your preferred web browser.
3. No build tools or package managers are required—just pure HTML, CSS, and JS!
